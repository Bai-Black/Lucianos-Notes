# Markdown 数学公式符号语法速查表

> 本文档汇总了在 Markdown 中使用 LaTeX 数学公式时的常用符号与语法。
> 行内公式用 `$...$`，独立公式用 `$$...$$`。

---

### 自适应括号大小

用 `\left` 和 `\right` 让括号自动匹配内容高度：

```latex
\left( \frac{a}{b} \right)        →  $\left( \frac{a}{b} \right)$
\left[ \sum_{i=1}^n x_i \right]   →  $\left[ \sum_{i=1}^n x_i \right]$
\left\{ x \mid x > 0 \right\}     →  $\left\{ x \mid x > 0 \right\}$
```

效果：$\left( \frac{a}{b} \right)$

```latex
\end{matrix}
$$
```

$$

$$

### 带括号的矩阵

```latex
% 圆括号
\begin{pmatrix} a & b \\ c & d \end{pmatrix}

% 方括号
\begin{bmatrix} a & b \\ c & d \end{bmatrix}

% 大括号
\begin{Bmatrix} a & b \\ c & d \end{Bmatrix}

% 行列式
\begin{vmatrix} a & b \\ c & d \end{vmatrix}

% 双竖线
\begin{Vmatrix} a & b \\ c & d \end{Vmatrix}
```

| 类型 | 语法 | 效果 |
|------|------|------|
| 圆括号 | `pmatrix` | $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$ |
| 方括号 | `bmatrix` | $\begin{bmatrix} a & b \\ c & d \end{bmatrix}$ |
| 行列式 | `vmatrix` | $\begin{vmatrix} a & b \\ c & d \end{vmatrix}$ |

### 省略号


## 15. 常用公式示例

### 二次方程求根公式

```latex
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
```

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

### 欧拉公式

```latex
e^{i\pi} + 1 = 0
```

$$e^{i\pi} + 1 = 0$$

### 勾股定理

```latex
a^2 + b^2 = c^2
```

$$a^2 + b^2 = c^2$$

### 泰勒展开

```latex
f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n
```

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$$

### 高斯积分

```latex
\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}
```

$$\int_{-\infty}^{\infty} e^{-x^2} \, dx = \sqrt{\pi}$$

### 矩阵乘法

```latex
\begin{pmatrix} a & b \\ c & d \end{pmatrix}
\begin{pmatrix} x \\ y \end{pmatrix}
= \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}
```

$$
\begin{pmatrix} a & b \\ c & d \end{pmatrix}
\begin{pmatrix} x \\ y \end{pmatrix}
= \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}
$$

### 贝叶斯定理

```latex
P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}
```

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

### 拉格朗日中值定理

```latex
\exists \, c \in (a, b), \quad f'(c) = \frac{f(b) - f(a)}{b - a}
```

$$\exists \, c \in (a, b), \quad f'(c) = \frac{f(b) - f(a)}{b - a}$$

## 17. 颜色
